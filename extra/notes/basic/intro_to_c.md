## **Intro to C**

---

**BASIC TYPES**

- char
- int
- float
- double
- short / long

strings are arrays of chars, ending in a null character
each char has its own memory address

---

**POINTERS**

- \* marks a variable as a pointer
- \*\* marks a variable as a pointer to a pointer
- malloc - allocates memory
- free - frees memory
